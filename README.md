# Mzansi Munch

A customer-to-customer food marketplace built for township entrepreneurs.

## About
Mzansi Munch connects informal food sellers with local buyers and delivery partners. 
Built with HTML, CSS, JavaScript, PHP, and MySQL.

## Live Site
https://mzansimunch.infinityfreeapp.com

## Features
- Role-based dashboards (Buyer, Seller, Delivery Partner, Admin)
- Escrow-style payment system
- Real-time order tracking
- Session timeout security
